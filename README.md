# Computaional Imaging

Self-contained implementations of some algorithms used in computational imaging, especially MRI. This repo has simple structure: one algorithm = one core python file + one demo notebook
```
algorithms
    |
    |- algorithm_1
    |   |- algo_1.py
    |   |- Demo.ipynb
    |
    |- algorithm_2
    |   |- algo_2.py
    |   |- Demo.ipynb
    ...
```


## Available Algorithms

- [SENSE reconstruction](algorithms/sense/)
- [CG-SENSE reconstruction](algorithms/cg_sense/)
- [Compressed sensing reconstruction](algorithms/cs/)
- [ESPIRiT calibration and reconstruction](algorithms/espirit/)
- [Extended phase graph simulation](algorithms/epg/)
